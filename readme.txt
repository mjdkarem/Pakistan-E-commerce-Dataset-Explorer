# Pakistan E-commerce Dataset Explorer

This is a Streamlit web application that performs exploratory data analysis (EDA) on Pakistan’s largest e-commerce dataset.  
It provides interactive dashboards, visual insights, and KPIs for business analysts, data scientists, e-commerce professionals, and startups.

---

## 🚀 Features
- Interactive KPIs: Total Revenue, Total Orders, Total Customers, AOV
- Visual Analysis: Monthly trends, category performance, customer behavior
- Deep Dive: Category-wise revenue heatmap
- Dark minimalist theme for better readability
- Integrated Tableau Dashboard for extended visualization

## 📁 Dataset Source
The dataset used in this project is from [Pakistan’s Largest E-commerce Dataset (Kaggle)](https://www.kaggle.com/datasets/zusmani/pakistans-largest-ecommerce-dataset).

## 🛠️ How to Run Locally
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/pakistan-ecommerce-eda.git
    cd pakistan-ecommerce-eda
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the app:
    ```bash
    streamlit run app.py
    ```
